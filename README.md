#Game Description

    This is an arcade-like game where one manuevers around a board
    
###Project Based On
    You may have to use CLion on MacOS to build this project.

##Dependencies
* [cmake](https://cmake.org/)

##Controls

###Keyboard

Key | Action
--- | ------
w | Move up
a | Move left
s | Move down
d | Move right
r | Reset game
q | Quit game
